# Form Filling Agent

A Claude Code SDK wrapper that automates fillable form completion using AI.

## Overview
A holiday Claude Code project that provides an extremely simple but useful app for automating PDF form filling. You upload a fillable form, provide context files, and chat with an agent to fill it out automatically.

## Key Features
- **Claude Code SDK integration** - Built on top of Claude Code's capabilities
- **LlamaParse integration** - Reliable document parsing without hallucinations
- **Multi-format support** - Handles messy scanned documents, photos, handwritten text
- **Interactive chat interface** - Chat with the agent to control form filling
- **Higher quality than alternatives** - Better and faster than ChatGPT/Claude UI out of the box

## Technology Stack
- Built with and powered by **Claude 3.5 Opus**
- Uses **LlamaParse** for document parsing
- Claude Code SDK wrapper
