## Claude Code

**Repository:** [anthropics/claude-code](https://github.com/anthropics/claude-code)

**By:** Anthropic

Claude Code is a powerful IDE-like interface for working with Claude AI. The tool recently released version 2.1.0 with significant improvements.

### Key Features (v2.1.0+):
- Shift+enter for newlines with zero setup
- Add hooks directly to agents and skills via frontmatter
- Skills: forked context, hot reload, custom agent support, invoke with /
- Agents no longer stop when tool use is denied
- Configure response language (Japanese, Spanish, etc.)
- Wildcard support for tool permissions
- Session teleporting capability
- Over 1096 commits in recent releases
