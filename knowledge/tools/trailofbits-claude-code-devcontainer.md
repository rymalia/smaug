# Trail of Bits - Claude Code DevContainer

## Overview
A sandboxed devcontainer setup for running Claude Code in bypass/YOLO mode safely. Built specifically for security audits and untrusted code review scenarios.

## Key Features
- **File Isolation**: Full file system sandbox
- **Network Isolation**: Network restrictions for safety
- **Security Audits**: Designed for reviewing untrusted code
- **Claude Code Compatible**: Works with Claude Code YOLO mode

## Use Cases
- Safely running Claude Code in bypass mode
- Auditing untrusted code
- Network and file system isolated development
