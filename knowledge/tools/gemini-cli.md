# Gemini CLI

AI command-line interface tool with built-in agent skill creation capabilities.

## Overview

Gemini CLI is a developer-friendly CLI tool that integrates Google's Gemini AI with a focus on creating and managing custom Agent Skills. The tool includes utilities for developers to write, test, and deploy custom skills for AI automation.

## Key Features

- **Built-in Skill Creator**: Automated skill generation based on examples and style guides
- **Agent Skill Support**: Create custom skills for AI agents to perform specialized tasks
- **Context-Aware Generation**: Understands patterns from provided examples and applies them consistently
- **Developer-Friendly**: Simple commands for skill creation and management

## Example Use Case

Jack Wotherspoon used Gemini CLI's skill-creator to:
1. Provided previous blog posts as examples
2. Provided a style guide for tone and formatting
3. Had the tool generate a blog-writer skill that follows the established patterns

This demonstrates how the tool can learn from context and generate production-ready skills without manual coding.
