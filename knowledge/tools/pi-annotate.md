# Pi Annotate

**Repository:** https://github.com/nicobailon/pi-annotate
**Stars:** 20
**Language:** JavaScript

## Overview

Pi Annotate is a Chrome/Pi browser extension that enables visual feedback from browser to AI. It allows developers to click on elements in the browser similar to web inspector functionality, add comments directly on elements, and have the agent receive CSS selectors, visual coordinates, and screenshots.

## Key Features

- **Element Selection:** Click on elements in your browser like the web inspector
- **Annotations:** Add comments directly to selected elements
- **AI Integration:** Sends selectors, CSS, and screenshots to AI agents
- **Visual Feedback:** Real-time visual indication of selected elements
- **Code Fixing:** Enables AI agents to understand and fix code based on visual inspection

## Use Cases

- Automated web testing and debugging with visual context
- AI-assisted web development and UI debugging
- Screenshot-based bug reporting with element identification
- Visual code repair workflows

## Tech Stack

- JavaScript
- Chrome Extension API
- Pi Framework integration
