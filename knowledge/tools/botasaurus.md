# Botasaurus

The all-in-one framework to build undefeatable web scrapers with advanced bot detection bypass capabilities.

## Overview

Botasaurus is a comprehensive web scraping framework that combines human-like behavior, bot detection evasion, and modern development practices to make building sophisticated scrapers easier and faster.

## Key Features

### Bot Detection Bypass

Proven to bypass industry-leading bot detection systems:
- Cloudflare Web Application Firewall (WAF)
- BrowserScan Bot Detection
- Fingerprint Bot Detection
- Datadome Bot Detection
- Cloudflare Turnstile CAPTCHA

### Core Capabilities

- **Undetectable Scrapers** - Human-like mouse movements and interaction patterns
- **Desktop Apps** - Convert scrapers into Mac, Windows, and Linux desktop applications
- **Web Interface** - Build beautiful websites for non-technical users to access scrapers
- **Cost Optimization** - Up to 97% savings on browser proxy costs using browser-based fetch requests
- **Parallel Scraping** - Easy asynchronous parallelization across multiple threads/machines
- **Utilities** - Built-in caching, sitemap support, data cleaning, proxy management
- **Kubernetes Scaling** - Scale scrapers across multiple machines with container orchestration
