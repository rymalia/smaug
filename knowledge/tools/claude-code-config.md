# Claude Code Config Repository

Open-source collection of Claude Code agents, commands, hooks, rules, skills, and plugins.

## Overview

Jarrod Watts has curated and created a collection of enhancements for Claude Code workflows. The repository includes:
- Custom agents for specialized tasks
- CLI commands and hooks
- Rules and constraints
- Skills for common operations
- Plugins for extending functionality

## Contents

All components are designed to be:
- Simple and straightforward
- Effective for real workflows
- Actively maintained and updated
- Community-friendly

## Use Cases

- Extending Claude Code capabilities
- Automating common development tasks
- Creating specialized agents
- Building custom workflows
