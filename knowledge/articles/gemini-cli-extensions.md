# Gemini CLI Extensions

## Overview

Gemini CLI now supports bundled extensions that combine multiple components for extending and customizing the Gemini CLI development experience.

## Supported Extension Components

- **Agent Skills:** Custom capabilities for AI agents
- **Hooks:** Lifecycle and event hooks for automation
- **MCP Servers:** Model Context Protocol server integrations
- **Context Files:** Custom context definitions
- **Custom Commands:** CLI command extensions

## Extension Gallery

Over 300+ extensions are currently available in the public gallery at https://geminicli.com/extensions/

## Getting Started

- Browse available extensions from the gallery
- Install extensions into your Gemini CLI setup
- Build custom extensions using the extension framework
- Share your extensions with the community
