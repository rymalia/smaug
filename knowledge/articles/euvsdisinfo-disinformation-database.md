# EUvsDisinfo: World's Largest Russian Disinformation Database

## Overview

EUvsDisinfo is the European Union's comprehensive open database documenting Russian disinformation operations. With nearly 20,000 cataloged cases accumulated over the past decade, it has become the authoritative reference for identifying and understanding Kremlin-originated false claims and propaganda campaigns.

## Key Information

- **Cases Documented:** 20,000+
- **Time Period:** 10 years of coverage
- **Geographic Focus:** EU, primarily
- **Primary Source:** Russian state and state-affiliated media narratives
- **Status:** Open to public access

## Purpose

The database serves as:
- A reference for identifying Russian disinformation narratives
- A research tool for studying propaganda patterns
- A public resource for media literacy and fact-checking
- A security resource for understanding information warfare tactics

## Access

The full database is available at: https://euvsdisinfo.eu/disinformation-cases

This represents a critical resource for anyone studying information security, geopolitics, or Russian information warfare strategies.
