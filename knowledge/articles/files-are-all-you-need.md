# Files Are All You Need

**Author:** LlamaIndex / Tuana Celik
**URL:** https://www.llamaindex.ai/blog/files-are-all-you-need
**Published:** January 2026

## Summary

Explores the evolving architecture patterns for AI agents, arguing that filesystems are increasingly adequate for many agent use cases instead of traditional databases. Demonstrates how Claude's agent skills feature (markdown directories) provide a simple, shareable alternative to server-based database infrastructure.

- **When Databases Still Win**: Complex queries, structured data, concurrent access requirements
