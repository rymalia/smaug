# Cursor's Dynamic Context Discovery

## Summary

Cursor has introduced dynamic context discovery for all models, making the system more intelligent about how context is filled while maintaining the same quality. This optimization reduces total token usage by 46.9% when using multiple MCP servers.

## Key Points

- **46.9% token reduction** when using multiple MCP servers
- **Intelligent context filling** maintains quality while reducing consumption
- **All models** benefit from the new dynamic context system
- Improvements in efficiency for integrated development workflows
